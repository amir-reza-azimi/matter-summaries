# matter-summaries

Static host for the **AI Video Digest**: listenable summaries of AI YouTube videos,
filtered for relevance to Amir's projects and published as an RSS feed that
[Matter](https://getmatter.com) subscribes to.

Generated by the `matter-video-digest` Claude Code skill (lives in the Obsidian vault).

- **Feed:** https://amir-reza-azimi.github.io/matter-summaries/feed.xml
- **Index:** https://amir-reza-azimi.github.io/matter-summaries/
- Per-video pages: `docs/videos/<youtube-id>.html`
- System of record for the feed: `data/<youtube-id>.json`

## Publish a summary

```bash
python3 scripts/publish_video.py record.json
```

`record.json` schema is documented at the top of `scripts/publish_video.py`. The
script writes the page, rebuilds `docs/feed.xml` and `docs/index.html` from every
`data/*.json`, and is safe to re-run (idempotent per video id).

GitHub Pages serves the `docs/` folder on `main`.
